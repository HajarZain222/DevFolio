# DevFolio

DevFolio is a responsive personal portfolio website designed to showcase personal information, skills, services, portfolio projects, blog posts, and contact details in a clean and professional layout.

## Features
- Fully responsive layout for all screen sizes
- Fixed navigation bar with multi-level dropdown menu
- Typing animation in the header section using Typed.js
- About section with personal details and skills progress bars
- Services section displayed using cards and icons
- Portfolio section with grid-based layout
- Testimonials slider using Bootstrap Carousel
- Blog section for articles and posts
- Contact form with social media links
- Font Awesome and Bootstrap Icons integration

## Built With
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Typed.js

## Project Structure
- `index.html` – Main HTML file
- `css/` – Stylesheets
- `js/` – JavaScript files
- `imgs/` – Images and assets

## Getting Started
1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Ensure all asset paths (CSS, JS, images) remain unchanged.

## Customization
- Update text content directly from the HTML file.
- Replace images inside the `imgs` folder.
- Modify styles from `style.css`.
- Adjust animations and interactions from `main.js`.

## Browser Support
Compatible with all modern browsers.
